# Leveraging-scHi-C-Data-for-Integrated-Single-Cell-Omics-Analysis

Code for: Leveraging scHi-C Data for Integrated Single-Cell Omics Analysis
This repository will host the official implementation for the paper, "Leveraging scHi-C Data for Integrated Single-Cell Omics Analysis", accepted at IEEE BIBM 2025.

Status
The code is currently being refactored, cleaned, and documented for public release.
We are working to ensure the code is user-friendly and reproducible. The full repository, including demo data and runnable scripts, will be made publicly available here ASAP.Thank you for your understanding.
